# ❌⭕ Tic Tac Toe Game in Java

A console-based **Tic Tac Toe Game** developed using **Object-Oriented Programming (OOP)** principles in Java. The application allows two players to compete on a 3×3 game board, validates player moves, detects winners, and identifies draw situations.

This project was developed as part of **Weekend Dev Challenge #57: LLD Projects** to strengthen Java programming, problem-solving, and software design skills.

# 🚀 Features

### 🎮 Two Player Gameplay

- Player X
- Player O

### 🎯 Interactive 3×3 Game Board

- Easy-to-understand console interface
- Displays board after every move

### ✅ Move Validation

- Prevents invalid positions
- Prevents overwriting occupied cells

### 🏆 Winner Detection

- Row Check
- Column Check
- Diagonal Check

### 🤝 Draw Detection

- Detects tie when the board is full

### 🔄 Turn-Based Gameplay

- Automatically switches turns between players

### ⚠️ Exception Handling

- Invalid input
- Already occupied position

### 📚 Clean Object-Oriented Design

# 🏗 Design Highlights

## Turn Management

The game alternates turns automatically between Player X and Player O after every valid move.

## Winner Detection

The system checks all rows, columns, and diagonals after each move to determine if a player has won.

## Board Validation

The application validates every move to ensure players cannot select an occupied or invalid position.
